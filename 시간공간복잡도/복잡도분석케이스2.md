# 코드 (투 포인터)

```python
class Solution:
    def isPalindrome(self, x: int) -> bool:
        s = str(x)  # O(n)
        left, right = 0, len(s) - 1
        
        while left < right:  # O(n)
            if s[left] != s[right]:
                return False
            left += 1
            right -= 1
        return True
```

## 간략히 분석

- 시간복잡도: while문에서 문자열 길이 n에 대해서 양쪽에서 한 번씩만 확인해서 n/2번 비교한다. 따라서 O(n)
- 공간복잡도: 입력 문자열 s 외에 추가 사용 메모리는 인덱스 left, right 2개 뿐이다. 따라서 O(1)
