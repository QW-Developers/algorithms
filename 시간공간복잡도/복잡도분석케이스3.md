# 코드(버블 정렬)

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):   # 매번 끝은 정렬 완료됨
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

## 간략히 분석
- 시간복잡도: for문이 중첩되어 보인다. 최악은 O(n^2), 최선은 이미 정렬되어 있을 경우 1번만 스캔하므로 O(n), 따라서 최악 O(n^2)
- 공간복잡도: 제자리에서 교체하는 정렬을 하고 있으므로 O(1)
